# EDGIE
This repository contains code that simulates the impacts of electric vehicles, heat pumps, and heat-pump water heaters on the distribution grid. It also includes a vehicle-to-home strategy to reduce the peak demand.

--------------------------------------------------------

Thank you for your interest in EDGIE!

INSTALLATION & DOCUMENTATION
------------

EDGIE needs MATLAB (https://www.mathworks.com/help/install/ug/install-products-with-internet-connection.html) and CVX (https://cvxr.com/cvx/download/) to run the files.

The repository includes sample weather files and commercial load profiles. However, these files for any specific county can be downloaded from https://data.openei.org/s3_viewer?bucket=oedi-data-lake&prefix=nrel-pds-building-stock%2Fend-use-load-profiles-for-us-building-stock%2F2025%2Fcomstock_amy2018_release_3%2F
